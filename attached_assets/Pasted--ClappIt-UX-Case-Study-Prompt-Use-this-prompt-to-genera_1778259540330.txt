# ClappIt UX Case Study Prompt

Use this prompt to generate a professional UX case study as if you worked as a UX Designer on the ClappIt platform in 2024.

---

Create a detailed and visually modern UX case study for a SaaS cloud infrastructure automation platform called ClappIt. The case study should be written in first person as if I worked as a UX Designer on this project during 2024.

Reference website: [ClappIt](https://clappit.io/?utm_source=chatgpt.com)

The tone should feel professional, strategic, and product-focused — similar to case studies on Behance, Medium, or top product designer portfolios.

# Case Study Structure

* Introduce ClappIt as an AI-powered cloud infrastructure and DevOps automation platform.
* Explain that the platform simplifies deployment workflows, CI/CD management, multi-cloud orchestration, infrastructure automation, and monitoring for engineering teams. ([sweya.ai][1])
* Mention that the goal was to reduce infrastructure complexity and improve operational efficiency for DevOps teams.

Include:

* Timeline
* Role: UX Designer
* Team collaboration
* Tools used (Figma, Miro, FigJam, Notion)
* Platforms designed (Web dashboard)
